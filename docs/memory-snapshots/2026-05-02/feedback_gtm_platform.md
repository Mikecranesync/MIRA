Do NOT default to YouTube Shorts as primary channel for B2B industrial/manufacturing content.

**Why:** ICP (plant managers, maintenance directors, Allen-Bradley users at 50–500 person manufacturers)
is on LinkedIn daily for professional content. They search YouTube for long-form technical tutorials
("Allen-Bradley VFD fault E005"), not 60s scroll. Shorts = top-of-funnel only.

**How to apply:** For any FactoryLM / MIRA content or GTM strategy:
1. LinkedIn native video = primary distribution
2. YouTube long-form = organic search capture
3. YouTube Shorts = algorithm seeding / derivative
4. TikTok/Reels = awareness only, low ICP conversion

Also: never use "11-year-old test" for this audience. Use "outsider engineer test" —
a competent engineer unfamiliar with the product should understand the value.
The ICP knows what "VFD overcurrent at 87Hz" means. Trust them.
