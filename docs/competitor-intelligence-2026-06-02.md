## Key Takeaways for FactoryLM / MIRA / CraneSync

1. **The namespace/grounding gap is real and unoccupied at SME price points.** Every enterprise player (Cognite, AVEVA, SparkCognition) solves this expensively. FactoryLM's $500 assessment → $499/mo path is genuinely differentiated.

2. **CraneSync's only true crane-specific incumbent is Konecranes** — and they only monitor their own equipment. OEM-agnostic crane AI (inspection scheduling + AI diagnostics + field service) has no direct startup competitor at scale.

3. **The $245M MaintainX raise defines the ceiling** for what the market will pay for a well-executed mobile-first CMMS. FactoryLM should compete on grounding and context, not feature parity.

4. **TRACTIAN ($196M) + Nanoprecise** define the sensor hardware path. FactoryLM's software-only (no proprietary sensors) approach is a deliberate wedge into brownfield sites that won't buy new hardware.

5. **OxMaint's knowledge-capture-from-work-orders** is the closest product-vision match to MIRA among bootstrapped players. Worth monitoring closely.
