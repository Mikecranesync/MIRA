## Target Personas

### Phase 1 (Users 1-5): "Floor Mike"

- **Title:** Maintenance technician or maintenance manager at a 10-100 person plant
- **Stack:** Allen-Bradley PLCs, VFDs (GS10/GS20/PowerFlex), possibly Ignition SCADA
- **Pain:** Buried in manuals, wastes 20-40 min per fault looking up codes
- **Watering holes:** YouTube (fault code searches), LinkedIn groups, r/PLC, r/maintenance, Facebook groups, TikTok (trades content)
- **Trigger phrase:** "Has anyone seen this fault code before?"

### Phase 2 (Users 6-10): "Integrator Ian"

- **Title:** Controls engineer or project manager at an Ignition certified integrator
- **Pain:** Customers ask support questions after commissioning
- **Trigger phrase:** "My customer keeps calling about the same alarms"

## Video Templates

### Long-Form: "AI vs. The Manual" (8-12 min)

```
[0:00-0:30]  HOOK - Face to camera at the panel
             "This VFD just threw an overcurrent fault. Let's see
             if AI can diagnose it faster than the manual."

[0:30-3:00]  THE PROBLEM - Show the fault on real equipment

[3:00-5:00]  THE MANUAL WAY - Pull out the physical manual or PDF.
             Time yourself finding the right page.

[5:00-7:00]  THE MIRA WAY - Screen recording.
             Type the same fault into Mira. Show the response.

[7:00-9:00]  COMPARISON - Side by side.
             Manual: X minutes. Mira: Y seconds.
             What it got right. What it missed. Be honest.

[9:00-10:00] THE FIX - Back on equipment. Show the resolution.

[10:00-11:00] CTA - Face to camera.
              "Try it free: app.factorylm.com"
```

### Short-Form: "Fault Code Quick Fix" (30-60 sec)

```
[0:00-0:03]  TEXT HOOK: "PowerFlex 525 F004 - check this FIRST"

[0:15-0:35]  Hands-on: "Here's what 90% of techs miss..."
[0:35-0:50]  Quick explanation of root cause + fix
[0:50-0:60]  "Full walkthrough on my channel. Link in bio."
```

## Risks & Mitigations

| Risk | Mitigation |
|------|------------|
| LinkedIn group is dead | Pivot to Reddit + personal LinkedIn by Day 5 if <50 views |
| Reddit removes self-promotion | Build karma first (Week 1 = zero links) |
| Low YouTube views early | Normal. Focus on search-optimized titles. Shorts carry discovery. |
| TikTok/Instagram feel like waste | Drop after 4 weeks if <500 views. Reallocate to DMs. |
| Academic outreach gets no replies | Follow up at 7 days. Try different professor at same school. |
| SBIR reauthorization delays | Draft Project Pitch now, submit day one when it reopens. |
| Content quality suffers | Cut to biweekly long-form if quality slipping. |
| Signups but no activation | DM every signup personally. Drip emails also handle this. |
