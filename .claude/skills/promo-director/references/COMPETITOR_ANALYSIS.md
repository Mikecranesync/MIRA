## Refresh delta — 2026-04-29

Initial population. No prior cache existed.

**New patterns worth watching:**

- **AI-agent reframe:** Both CMMS (UpKeep Nova) and dev-tools (Cursor 3) vendors are abandoning feature-list promos in favour of "your always-on AI teammate / built for a world where all code is written by agents" framing. Functional claim → agent relationship claim.
- **Fear hook + dollar number → credibility:** Tractian's dominant video hook anchors on "$1.4 trillion in annual downtime costs" before introducing product. This is the clearest proof template in the industrial segment.
- **Category expansion as launch event:** Augury's Ultra Low RPM launch (March 2025) and MaintainX's OT Data Connectors (December 2025) both use product launches to announce TAM expansion, not just feature adds. PR-driven, minimal YouTube presence.
- **Figma's philosophical hook:** "The future won't be designed by accident" (Config 2025) — opens with world-view statement, not product claim. Rare in B2B but high-signal for premium positioning.
- **Cursor 3 declarative simplicity:** Three-clause positioning ("simpler, more powerful, built for a world where all code is written by agents") used consistently across video title, X post, and blog. Replicable template for MIRA.
- **Stripe scale + AI combo:** "50,000 transactions/minute" + new AI features (Smart Disputes, Authorization Boost) in same keynote. Pairs trust-signal (scale number) with future-signal (AI).
- **MaintainX velocity signal:** 3 major product releases in Q1 2026 alone (OT connectors Dec '25 → Smart Tag Mapping Mar '26 → Report Builder AI Apr '26). No major brand video; they're shipping fast and blogging it, not filming it.

## Anti-patterns observed

1. **Tutorial as brand content:** CMMS vendors (eMaint, Fiix, Limble) use tutorial videos as primary YouTube presence — these convert existing evaluators but don't create category demand.
2. **Conference keynote as only launch video:** Figma, Stripe, OpenAI all rely on keynote formats that are too long for cold audiences. Best in class also ship a compressed summary video (Figma does; not all do).
3. **No video at all for major launches:** Augury ($75M round, Ultra Low RPM launch), MaintainX (3 major Q1 2026 releases) — significant news with zero YouTube content. Signal: these brands are betting on PR and analyst channels, leaving a video gap anyone can fill.
4. **Acquisition quiets output:** Loom post-Atlassian acquisition has reduced distinctive video presence.
