**Frame:** Solid dark background, single URL, nothing else
**Narration:** `<price anchor + trial framing + URL>`
**Validation:**
- [ ] One URL only
- [ ] Price anchor BEFORE trial framing
- [ ] URL on screen for 6 full seconds
- [ ] No alternate CTAs
