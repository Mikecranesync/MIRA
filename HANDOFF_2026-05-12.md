## Key Lesson (write to memory next session)

The 2026-04-25 overnight regression and the 2026-05-11 17-hour run had the **same root cause**: `tools/hooks/stop-gate.sh` was built and committed but never wired into `.claude/settings.json`. Building safety infrastructure ≠ activating it. **Anytime a new hook/gate script is added, the same PR must also wire it into the settings.json that actually invokes it** — otherwise it's dead code.
