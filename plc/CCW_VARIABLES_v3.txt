NEW VARIABLES (add these in CCW Global Variables):

  Name                  Type    Initial   Purpose
  --------------------  ------  --------  -----------------------------------
  dir_fwd               BOOL    FALSE     Selector in FWD position
  dir_rev               BOOL    FALSE     Selector in REV position
  dir_off               BOOL    FALSE     Selector in OFF position
  dir_fault             BOOL    FALSE     Invalid selector state (both closed)
  estop_wiring_fault    BOOL    FALSE     E-stop dual-channel XOR violation
  prev_button           BOOL    FALSE     RUN pushbutton previous state
  vfd_poll_active       BOOL    FALSE     Modbus message in progress
  vfd_poll_step         INT     0         Current poll step (1=read, 2=write_cmd, 3=write_freq)
  vfd_freq_setpoint     INT     0         Calculated VFD frequency (0-400)
  vfd_cmd_word          INT     5         VFD command register value (5=stop)

NEW MSG INSTANCES (for VFD write commands):

  Name                  Type    Purpose
  --------------------  ------  -----------------------------------
  mb_write_cmd          MSG     Write command word to VFD reg 0x2100
  mb_write_freq         MSG     Write freq setpoint to VFD reg 0x2101

NEW CONFIG STRUCTS:

  Name                     Type              Purpose
  -----------------------  ----------------  -----------------------------------
  write_cmd_local_cfg      MSG_MODBUS_LOCAL  Config for command write
  write_cmd_target_cfg     MSG_MODBUS_TARGET Config for command write
  write_freq_local_cfg     MSG_MODBUS_LOCAL  Config for freq write
  write_freq_target_cfg    MSG_MODBUS_TARGET Config for freq write

NEW DATA ARRAYS:

  Name                  Type        Purpose
  --------------------  ----------  -----------------------------------
  write_cmd_data        INT[1..10]  Buffer for command write
  write_freq_data       INT[1..10]  Buffer for freq write
