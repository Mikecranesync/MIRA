# Skill: Cold Outreach Drafting

## Trigger
"Draft outreach to [company/person]", "write cold email", "reach out to [prospect]"

## What It Does
1. Research the target company and person (web search + any enrichment tools)
2. Read ABOUT_MIRA.md and ICP_PROFILES.md for positioning context
3. Identify which ICP the prospect matches and select appropriate messaging
4. Draft a personalized cold email using the relevant persona messaging from ICP_PROFILES.md
5. Include a specific pain signal or trigger event if found in research
6. Save draft to PROSPECTS/ folder and optionally create Gmail draft

## Output Format
Save to: `PROSPECTS/[company_name]/outreach_[date].md`

```
# Outreach: [Company] — [Contact Name]

## Research Summary
- Company: [overview]
- Contact: [title, background]
- Pain signal: [what triggered this outreach]
- ICP match: [which ICP profile]

## Email Draft
Subject: [subject line]
---
[email body]
---

## Follow-up Plan
- Day 3: [action]
- Day 7: [action]
- Day 14: [action]
```

## Connected Tools Needed
- Web search (account research)
- Gmail (create_draft)
- File system (read ICP_PROFILES.md, ABOUT_MIRA.md)
