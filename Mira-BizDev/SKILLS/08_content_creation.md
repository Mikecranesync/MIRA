# Skill: Sales Content & Asset Creation

## Trigger
"Write a [one-pager/case study/LinkedIn post/email sequence]", "create sales content", "I need a [asset type] for [audience]"

## What It Does
1. Read ABOUT_MIRA.md for product context and positioning
2. Read ICP_PROFILES.md to match content to the right audience
3. Research the topic or audience segment as needed
4. Draft the requested asset with MIRA's voice and messaging
5. Save to ASSETS/ folder

## Asset Types
- **One-pager:** Single-page PDF-ready overview for a specific ICP
- **Case study:** Before/after story (even hypothetical at pre-revenue stage)
- **LinkedIn post:** Thought leadership content about industrial maintenance + AI
- **Email sequence:** Multi-touch nurture sequence for a specific persona
- **Demo script:** Guided walkthrough for live demos
- **ROI calculator narrative:** Downtime cost math + MIRA payback framing
- **Trade show talking points:** Quick reference for in-person conversations

## Output Format
Save to: `ASSETS/[asset_type]_[audience]_[date].md`

## Voice Guidelines
- Practical, not hype-y. MIRA is built by someone who understands the floor.
- Use specific technical examples (fault codes, equipment models) — not generic AI buzzwords.
- Lead with the pain, not the technology.
- Short sentences. No jargon the buyer wouldn't use themselves.
- Always include a concrete number or scenario (downtime cost, time saved, knowledge retained).

## Connected Tools Needed
- File system (ABOUT_MIRA.md, ICP_PROFILES.md)
- Web search (for industry stats, examples)
